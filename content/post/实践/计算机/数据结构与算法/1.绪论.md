各个问题
1. 如何求解问题
	问题 -> 抽象化 -> 数据结构+算法 -> 程序
2. 解决问题的方法效率与什么有关
	1. 数据的组织方式
	2. 空间利用率
	3. 算法
3. 什么是数据结构？
	1. 相关概念
		1. 数据对象
			实际问题中计算机要处理的对象（被输入到计算机，并由计算机处理的符号的集合）
		2. 数据元素
			数据对象(集合）中的一个对象，比如一个学生
		3. 数据项
			一个数据元素所具有的数据，比如一个学生的成绩或者身高、性别等
	2. 定义
		数据结构指 数据对象在计算机中的组织方式，可定义为 $(D,S)$ ，$S$ 是数据对象 $D$ 上的一个有穷关系
		- 线性结构
		- 树
		- 图
		- 集合
	3. 存储结构
		1. 二进制存储
		2. 存储关系
			1. 顺序存储
			2. 链式存储（链表）
		3. 存储数据时，同时存储数据的值和关系
		4. ==注意，数据的逻辑结构独立于其存储结构，我们所学的数据结构都属于逻辑结构，存储方式本质相同==
4. 抽象数据问题
	1. 定义
		1. 数据类型
		 数据对象的类型，包含其定义域、操作集
		2. 抽象数据类型
		 一个数学模型的定义和其之上的一组操作
5. 算法的概念、特性和设计原则
	1. 概念：为了解决某个问题而采取的一系列操作
	2. 特性
		1. 有穷
		2. 确定
		3. 可行
		4. 有输入
		5. 有输出
	3. 设计原则
		1. 正确
		2. 可读
		3. 健壮
		4. 高效
		5. 低存储量（相对于要求）
6. 空间复杂度分析
	数据数量 $\times$ 单个数据所占空间
7. 时间复杂度分析
	1. 时间复杂度函数
		1. 渐近上界
		2. 渐近下界
		3. 渐近确界
	2. 常见时间复杂度
		- $n$
		- $n^2$
		- $\log n$
		- $2^n$
	3. 常见计算
		1. 循环嵌套
			乘法
		2. 选择结构
			取最大值
		3. 递归
			- 推导递归方程
	4. 使用 clock() 函数来获取程序的实际运行时间
