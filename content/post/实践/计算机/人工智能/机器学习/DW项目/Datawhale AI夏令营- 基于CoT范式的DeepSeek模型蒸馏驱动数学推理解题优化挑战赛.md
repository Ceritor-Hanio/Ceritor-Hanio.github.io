本赛事以高等数学推理为核心任务，聚焦模型的“解题过程”生成能力，强调使用 **CoT（Chain-of-Thought）思维链范式** 来提升模型对复杂数学题的理解与推理能力。

需要完成的任务：
- 基于提供的开源高等数学题数据集，
- 通过**模型蒸馏**
- 训练出一个带CoT的学生模型

## 1. 数据分析
官方给了两个数据集
1. 训练集
	有问题，有答案。
	 ```json
	 {
    "id": "564",
    "input": "男生比女生多60%，女生比男生少多少%．",
    "output": "如果男生比女生多60%，那么男生人数是女生人数的160%（100% + 60%）。 \n要求女生比男生少多少%，可以计算出女生人数与男生人数的比值。 \n女生人数/男生人数 = 1/1.6 = 0.625\n这意味着女生人数是男生人数的0.625倍。\n要计算女生比男生少多少%，需要计算（男生人数-女生人数）/男生人数 × 100% ：\n(1-0.625)/1 × 100% = 0.375 × 100% = 37.5%\n女生比男生少37.5%。"
}
	 
	 ``` 

2. 测试集
	只有问题，用于测试模型的蒸馏效果。

## 2. 模型蒸馏的过程
1. 人工清洗数据集
2. 使用训练集微调教师模型
3. **教师模型** 对 **测试集** 进行 **批量推理**，获得包含完整CoT推理链的数学解题输出 **微调集**
4. 选择**学生模型**，使用 **微调集** 进行 **LoRA 精调（即微调）**
