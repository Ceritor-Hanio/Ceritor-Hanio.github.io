**研究力和运动关系**

## 1.1 质点运动的描述

1. 运动的描述是相对的——**要选择参考系**
	一个固定在参照物上的坐标系和相应的一套同步时钟组成一个参考系
2. **位置矢量**  $\vec{r}$ 
	运动的质点相对于原点的位置
	- 位置函数 $\vec{r}=\vec{r}(t)=\left\{\begin{array}\ x=x(t)\\y=y(t)\\z=z(t)\end{array}\right.$
	- 位移 $\Delta\vec{ r}=\vec{r}(t+\Delta t)-\vec{r}(t)=\Delta x \vec{i}+\Delta y \vec{j}+\Delta z \vec{k}$
	- 轨迹方程 $f(\vec{r})=0$
	
	- 加速度 $\vec{a}=\frac{{d\vec{v}}}{dt}$
3. 速度
	- 平均速度 $\bar{\vec{v}}=\frac{\Delta r}{\Delta t}$
	- 速度 $\vec{v}=\frac{{d\vec{r}}}{dt}$
		- 速率 $v=|\vec{v}|=|\lim_{ t \to 0 } \frac{\Delta\vec{r}}{\Delta t}|=|\lim_{ t \to 0 } \frac{\Delta s}{\Delta t}|=\frac{ds}{dt}$
	- 平均速率
		$\bar{v} = \frac{\Delta s}{\Delta t}$
4. 加速度
	- 平均加速度
		- $\bar{a}=\frac{\Delta v}{\Delta t}$
		- $\vec{a}=\lim_{ \Delta t \to 0 }  \frac{\Delta\vec{v}}{\Delta t}=\frac{d \vec{v}}{dt}$

## 1.2 匀变速运动 抛体运动

### 1.2.1 匀变速运动
加速度是常矢量： $\vec{a}=\vec{c}$
1. $\vec{a}=\frac{{d \vec{v}}}{dt}\implies    \vec{v}=\vec{v_{0}}+\vec{a}t$
2. $\vec{r}=\frac{{d \vec{r}}}{dt}\implies  \vec{r}$

### 1.2.2 抛体运动

2. 沿原运动方向和竖直方向分解

## 1.3 圆周运动

### 1.3.1 圆周运动描述
1. 角量描述
	1. $\Delta \theta=\theta(t+\Delta t)-\theta(t)$
	2. $\bar{\omega}=\frac{\Delta \theta}{\Delta t}$
	3. 角速度 $\omega=\lim_{ \Delta t\to 0 } \frac{{\Delta \theta}}{\Delta t}=\frac{d\theta}{dt}$
	4. 角加速度 $\alpha=\frac{d\omega}{dt}=\frac{d^2\theta}{dt^ 2}$
2. 自然坐标系
	1. 分成切向 $\vec{\tau}$ 和法向 $\vec{n}$ 
	2. $\vec{v}_{n}=0,\vec{v}=\vec{v}_{t}=v\vec{\tau}$
3. $v=\omega R$
4. 加速度
	$\alpha=\frac{d\vec{v}}{dt}=\frac{d(v \vec{\tau})}{dt}$
	1. 切向加速度
		$\alpha_{\tau}=\frac{dv}{dt}=\alpha R$
	2. 法向加速度
		$\vec{a}_{n}=v\omega  \vec{n}=R\omega^2 \vec{n}$

### 1.3.2 平面极坐标系
- 径向单位向量 $\vec{e}_{r}$
- 横向单位向量 $\vec{e}_{\theta}$
- 位矢 $\vec{r}=r(t)\vec{e}_{r}$
- 速度 
	$\vec{v}=\frac{{d \vec{r}}}{dt}=\frac{{d r(t)\vec{e}_{r}}}{dt}=\vec{e}_{r}r'(t)+r(t)(\vec{e}_{r})'$
	- $\frac{{d\vec{e}_{r}}}{dt}=\frac{{d\theta}}{dt}\vec{e}_{\theta}$
	- $\frac{{d\vec{e}_{\theta}}}{dt}=-\frac{{d\theta}}{dt}\vec{e}_{t}$
	$$\implies   \vec{v}=v_{r} \vec{e}_{r}+v_{\theta}\vec{e}_{\theta}$$

## 1.4 相对运动
- 伽利略速度变换
	- 质点相对静系的速度 = 动系相对静系的速度 + 质点相对于动系的速度
- 加速度同理
